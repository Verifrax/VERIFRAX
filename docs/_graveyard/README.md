# Non-Authoritative Documentation

This directory contains documentation that is **not part of VERIFRAX v2**.

These documents are preserved for historical reference but are:
- Not frozen
- Not normative
- Not referenced by frozen governance documents
- Not part of the v2 trusted computing base

## Authoritative Documentation

Only the following documents in `docs/` are authoritative for v2.0.0:

- `V2_EDGE_API.md` - API specification
- `DNS_AUDIT_v2.md` - DNS security audit
- `GOVERNANCE_DISPUTE_FINALITY.md` - Dispute protocol
- `GOVERNANCE_VERSION_FINALITY.md` - Version guarantees
- `EVIDENCE_BUNDLE_SPEC_v1.md` - Bundle specification
- `PRICING_RISK_TIERS.md` - Pricing structure

All other documentation is archived here.
