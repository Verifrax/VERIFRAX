# Code of Conduct

## Our Standards

VERIFRAX is a technical project focused on deterministic verification. Interactions should be:

- **Technical**: Focus on code, documentation, and specifications
- **Constructive**: Propose solutions, not just problems
- **Respectful**: Disagree with ideas, not people
- **Concise**: Value everyone's time

## Expected Behavior

- Use clear, professional language
- Provide context and reproduction steps for issues
- Accept constructive feedback
- Focus on what is best for the project

## Unacceptable Behavior

- Personal attacks or insults
- Harassment of any kind
- Publishing others' private information
- Spam or off-topic content
- Demanding features or timelines

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

## Reporting

Report violations to: conduct@verifrax.net

Reports are reviewed within 7 days.

## Scope

This code of conduct applies to:
- GitHub issues and pull requests
- Project discussions
- Any project-related communication
