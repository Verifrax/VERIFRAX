# Scope

## Hard inclusions

- Deterministic verification of evidence bundles
- Single-execution certificate issuance
- Hash-addressable certificate storage
- Offline certificate verification
- Payment-gated execution

## Hard exclusions

- Data storage or hosting
- Human review or judgment
- Certificate amendment or revocation
- Re-execution of same bundle
- Content quality assessment
- Legal determination
- Continuous monitoring
- Batch processing
- Mutable claims

## Boundary conditions

- One payment → one token → one execution → one certificate
- Certificate validity independent of infrastructure
- No state persists beyond certificate issuance
- No external dependencies for verification

## Non-goals

- General-purpose timestamping
- Blockchain notarization
- Document signing service
- Audit trail system
- Compliance certification
